# 🌿 Calmora — UX Case Study

**A habit-building yoga experience designed to improve consistency, motivation, and mindfulness through structured tracking and simple interaction design.**

---

## 👩‍💻 About the Project

Calmora is a mobile UI/UX case study focused on designing a habit-forming wellness experience for yoga practitioners.  
The goal was to reduce inconsistency in daily yoga practice by creating a minimal, guided, and motivating user experience.

---

## 🎯 Problem Statement

Most fitness and wellness apps fail to retain users due to:

- Lack of habit reinforcement systems  
- Low motivation and engagement over time  
- Complex onboarding flows  
- No clear progress visualization  

---

## 🎯 Design Goal

To design a user-centered yoga habit tracking experience that:

- Encourages daily consistency  
- Improves user motivation through visual progress  
- Reduces cognitive load in onboarding and tracking  
- Provides a clean and distraction-free interface  

---

## 🧠 UX Process

User Research → Personas → Problem Definition → User Flows → Wireframing → UI Design → Prototyping

This project follows a complete end-to-end UX design process focused on real user behavior and usability principles.

---

## 🧩 Key Features Designed

- Daily yoga habit tracking system  
- Streak and progress visualization  
- Simple and guided onboarding flow  
- Minimal and calm interface design  
- Motivation-driven user experience  

---

## 🎨 UX Principles Applied

- Information Architecture  
- Visual Hierarchy  
- Cognitive Load Reduction  
- Behavioral Design Thinking  
- Usability Heuristics  
- Mobile-First Design Approach
